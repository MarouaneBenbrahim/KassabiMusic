# KASSABI Website

Production-ready static website build in `web/`.

## Run locally

```powershell
python -m http.server 8080 --directory web
```

Open `http://localhost:8080`.

## Deploy

Deploy the full `web/` folder to any static host:

- Cloudflare Pages
- Netlify
- Vercel (static)
- GitHub Pages

No build step is required.
